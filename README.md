# AI自己理解ラボ Web診断

このリポジトリは、AI自己理解ラボの「あなたの本音タイプ診断」を公開するためのGitHub Pages用ファイルです。

## 公開ページ

`honne-diagnosis/`

## 構成

```text
honne-diagnosis/
  index.html
  style.css
  script.js
assets/
  ai-self-lab-icon.png
```

## GitHub Pages設定

GitHubの `Settings` → `Pages` で以下を設定します。

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

公開後のURL例:

`https://ユーザー名.github.io/リポジトリ名/honne-diagnosis/`
