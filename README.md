# AI自己理解ラボ Web診断

このリポジトリは、AI自己理解ラボの「あなたの本音タイプ診断」を公開するためのGitHub Pages用ファイルです。

## 公開ページ

`https://ai-self-lab.com/`

## 構成

```text
index.html
style.css
script.js
assets/
  ai-self-lab-icon.png
CNAME
```

## GitHub Pages設定

GitHubの `Settings` → `Pages` で以下を設定します。

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

公開後のURL例:

`https://ai-self-lab.com/`
